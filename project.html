Features That I Made In This Project

  • Web Login
    1. Gather the username and password from the user.
    2. Encrypted the password using a secure encryption algorithm (bcrypt).
    3. Verified the username and encrypted password against the stored values in the database.
    4. If the username and password match, grant access to the website; otherwise, display an error message.
    5. Designed and code the ui of the web login
  • Mobile Login
    1. Gather the username and password from the user.
    2. Encrypted the password using a secure encryption algorithm (JWT).
    3. Verified the username and encrypted password against the stored values in the database.
    4. If the username and password match, grant access to the website; otherwise, display an error message.
    5. Designed and code the ui of the mobile login
  • Mobile Logout
    1 Removed the current session of the JWT User Token
    2 Designed and code the user interface (UI) of the mobile logout
  • Appointment
    1 Designed the user interface (UI) to include the appointment form with doctor selection and scheduling validation.
    2 Implemented the UI design using Flutter.
    3 Developed the backend code to handle the appointment submission, doctor schedule validation, and database operations if required.
  • Sign Up Mobile
    1 Designed the user interface (UI) to include the signup form with name and password fields, along with password validation.
    2 Implemented the UI design using Flutter.
    3 Developed the backend code to handle the signup form submission, perform password validation, and store user information in a database if required.
  • Membuat Chart Total Penjualan Dan Pendapatan Obat
    1 Developed the backend code to get the correct data
    2 Implemented the UI design using HTML CSS AND JAVA SCRIPT and Chart.js Library.
